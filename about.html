Acerca De Este Sitio

Este sitio web fue creado a raiz de un proyecto del curso "Desarrollo Web". Este sitio web funciona como un portafolio virtual en linea para acceder con facilidad a los aspectos generales de mi persona. Honestamente, varias paginas que desarrolle probablemente las descarte. Sin embargo, hay algunas paginas que me gustaria dejar e incluso refinar a un nivel mayor para aprender y demostrar conocimientos de desarrollo web, asi elevar mi portafolio virtual y demostrar mi valor como desarrollador. Es por esta razon que este ejercicio se me ha hecho de gran valor y me ha despertado mas interes por desarrollar paginas web. Lastima que por tiempo no pude tomarme mas libertades creativas.

Tecnologias Utilzadas

  • Bootstrap: Se ha utilizado los componentes de Bootstrap para desarrollar la pagina web. Tambien se han utilizados algunos iconos ofrecidos por Bootstrap.
  • Font Awesome: Se han utilizado varios iconos de Font Awesome, como en la pagina de "Por que programacion?", en la cual se implemento iconos utilizando el feature de "stack"
